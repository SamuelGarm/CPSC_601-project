Controls:
	Mouse wheel = zoom camera
	RM and drag = pan camera
	
	Simulation speed is in FPS, 1 earth day = 60 frames as a reference for speed adjustments

Simulation is set up to run at a max of 60FPS. If this is run on a potato the framerate may be slower but I'm pretty sure you will be fine
